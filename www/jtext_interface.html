Textchecker

Geben Sie hier Ihren Text ein (das Tool kann aktuell nur deutsche Texte verarbeiten). Wenn Sie auf "Prüfe meinen Text" klicken, erhalten Sie einen Überblick, ob wichtige Kriterien transparenter Berichterstattung als erfüllt klassifiziert werden können. Der Fokus liegt dabei auf transparenten Zahlenformaten (s. ).

Bitte beachten Sie: Das Tool wird manche Zahlen nicht korrekt identifizieren (insbesondere, wenn Texte die keine Interventionen, wie Medikamente oder Impfungen behandeln eingegeben werden).

Falls etwas nicht funktioniert, können Sie uns hier eine kurze Rückmeldung geben (dabei wird der Text per Email an risk.booster@gmail.com geschickt, damit wir den Fehler beheben können):
