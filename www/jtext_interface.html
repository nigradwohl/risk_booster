Textchecker


Geben Sie hier Ihren Text ein (das Tool kann aktuell nur deutsche Texte verarbeiten). Wenn Sie auf "Prüfe meinen Text" klicken, erhalten Sie einen Überblick, ob wichtige Kriterien transparenter Berichterstattung als erfüllt klassifiziert werden können (s. Wiki). Die entsprechenden Zahlen werden im Text hervorgehoben.

Bitte beachten Sie: Das Tool wird manche Zahlen nicht korrekt identifizieren (insbesondere, wenn Texte, die keine Interventionen, wie Medikamente oder Impfungen, behandeln, eingegeben werden).

Ihr Text und Ihre Anmerkungen werden lokal verarbeitet und nicht auf Servern gespeichert.
